Installation
################


Pypi
**********************

.. code-block:: console

    # Install from Pypi:
    pip install thompson

    # Force update to latest version
    pip install -U thompson


Github source
************************************

.. code-block:: console

    # Install directly from github
    pip install git+https://github.com/erdogant/thompson


Uninstalling
################


Remove installation
**********************

Note that the removal of the environment will also remove the ``thompson`` installation.

.. code-block:: console

    # Install from Pypi:
    pip uninstall thompson
